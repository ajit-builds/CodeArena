🎮 CodeArena – Coding Challenge Game Arena

An interactive web-based coding practice platform designed to help beginners learn programming concepts through gamified challenges and mini-games.

The platform provides multiple coding games, scoring systems, timers, and a leaderboard to encourage consistent learning and competition.

🚀 Features

🎮 Gamified Coding Challenges

⏱ Timed Programming Tasks

🧠 Multiple Learning Games

🏆 Leaderboard Ranking System

🎯 Difficulty Levels (Easy / Medium / Hard)

🛠 Game Builder for Custom Challenges

💾 Local Storage Score Tracking

🎨 Arcade-style Developer UI

🕹 Included Coding Games
1️⃣ Coding Hangman

Guess programming-related words based on hints.

Topics:

HTML

CSS

JavaScript

C++

2️⃣ Syntax Reveal

Identify missing or incorrect syntax in code snippets.

3️⃣ MCQ Coding Quiz

Multiple-choice questions that test programming fundamentals.

4️⃣ CSS Tic Tac Toe

A layout-based game designed to teach Flexbox and Grid concepts.

🛠 Tech Stack

HTML5 – Page structure

CSS3 – UI design and layout

JavaScript – Game logic and interactions

LocalStorage – Score tracking and leaderboard

VS Code – Development environment

📊 How It Works

User opens the CodeArena platform

Selects a coding challenge from the game library

Completes the challenge within the time limit

The system validates the response

Score is calculated based on performance

Leaderboard updates with the new ranking

🎯 Objective

The goal of this project is to demonstrate how gamification can enhance programming education by combining coding challenges with engaging gameplay elements.
