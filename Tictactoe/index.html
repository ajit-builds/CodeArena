// HOW TO PLAY

01
Look at the 9 cells on the board. Each cell has a CSS coordinate label.
02
Type the matching CSS values in the terminal on the left. The target cell will highlight.
03
Press ▶ Execute Placement (or Enter twice) to place your mark on the targeted cell.
04
Get 3 in a row before your opponent. In Flexbox mode use justify-content + align-items. In Grid mode use grid-column + grid-row.
05
Win quickly with fewer wrong inputs to earn a higher score. Scores sync to the Leaderboard page.
